샘플 결과물

# 회의요약 Skill - 언제 쓰나: 회의 메모가 길고 복잡할 때 - 잘하는 것: 핵심 요약, 액션아이템 정리, 담당자/기한 추출 - 답변 방식: 3줄 요약 → 액션아이템 표 → 빠진 정보 질문 1개 - 하지 말 것: 없는 결정사항을 지어내지 않기
프롬프트 예시

Claude Code에 이렇게 말해보기

회의요약 Skill 초안을 만들어줘. 이 Skill은 긴 회의 메모를 입력받아 1) 3줄 요약 2) 액션아이템 표 3) 빠진 정보 질문 1개 형태로 답하게 해줘. 과장하거나 없는 결론은 만들지 않게 해줘.
