샘플 결과물

이런 출력이 나오면 좋습니다

1. Hero: Claude Code로 강의자료를 더 빨리 만드는 법 2. 오늘 배우는 것: 자동화 흐름 3단계 3. 핵심 개념: 입력자료 / 출력형식 / 수정프롬프트 4. 실습 1: 주제와 학습목표 넣기 5. 실습 2: 슬라이드 구조 생성 6. 결과물 예시: 슬라이드 개요, 워크북 문안, 퀴즈 3문항 7. 치트시트: 바로 쓸 프롬프트 5개
왜 이 명령어가 2주차에 많이 쓰이나

2주차 자주 쓰는 명령어 + 이유 + 예시 + 프롬프트

명령어왜 2주차에 많이 쓰이나실제 예시Claude Code 프롬프트 예시
claude -p "질문"강의구조, 개요, 초안 문안을 빠르게 한 번 뽑아볼 수 있어서강의 주제 한 줄로 개요 초안 생성
claude -p "간호대 학생 대상 AI 입문 강의 개요를 7개 섹션으로 짜줘"
@./src/기존 페이지 파일을 정확히 가리켜서 수정 방향을 줄 수 있어서이미 만든 웹페이지 구조를 강의형으로 바꾸기
@./src/ 이 페이지를 강의 안내형 구조로 바꾸고 Hero, 실습, 결과물 섹션을 먼저 제안해줘
--add-dir강의자료, 이미지, 참고문서가 여러 폴더에 흩어져 있을 때 함께 보게 할 수 있어서slides 폴더와 docs 폴더를 동시에 참조
claude --add-dir ./slides ./docs "두 폴더를 참고해서 강의자료 앱 구조를 제안해줘"
!npm run build페이지 초안을 만든 뒤 실제로 깨지지 않는지 바로 확인할 수 있어서수정 후 빌드 확인
페이지 초안 수정 후 !npm run build 해보고 에러 있으면 같이 고쳐줘
/compact강의자료 논의가 길어지면 핵심만 남기고 이어가야 해서슬라이드 구조 논의 후 압축
/compact 그리고 지금까지 정한 구조를 바탕으로 실습 섹션만 이어서 만들어줘
